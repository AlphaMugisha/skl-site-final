# NGA Coding Academy Website

## Project Overview

This is a **static multi-page website** for **NGA Coding Academy**, a private coding and technology school located in Kigali, Rwanda.  
The goal of this project is to create a **modern, professional, and responsive website** for the academy that is easy to maintain and visually appealing.

## Website Structure

The website consists of **five pages**:

1. **Home** – Hero section, programs overview, levels, facilities, call-to-action.  
2. **About Us** – Academy introduction, mission & vision, why choose NGA Coding Academy, facilities description.  
3. **Academics** – Programs offered, levels explanation, learning approach.  
4. **News** – School news and updates layout.  
5. **Contact** – Contact details, contact form, map placeholder.

All pages are **HTML files** with a **consistent header** including:

- Home  
- About Us  
- Academics  
- News  
- Contact  

The header is **sticky, responsive, and mobile-friendly**.

---

## CSS & Design

The CSS was created to ensure:

- **Modern, professional, and clean layout**  
- **Simplicity and readability**  
- Rounded cards, soft shadows, and hover effects  
- Consistent spacing, padding, and margins  
- Clear typography and visual hierarchy  
- Mobile responsiveness for all sections  

### Improvements & Inspiration

- Original CSS was basic. It was **improved for simplicity and readability**.  
- Design inspiration was taken from **New Generation Academy**, including color schemes, typography, layout, and UI/UX enhancements.  
- Subtle animations and hover effects were added to buttons and cards.  
- Navigation bar is sticky, responsive, and adapts to mobile screens.

## How to Use
